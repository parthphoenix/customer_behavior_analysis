# Customer Shopping Behavior Analysis

## Overview

This project analyzes **customer shopping behavior using transactional purchase data** to uncover insights about spending patterns, product preferences, customer segments, and subscription behavior.

The project demonstrates a complete **data analytics workflow**, including data cleaning in Python, SQL-based business analysis, and visualization using Power BI. The final insights are presented through a report and presentation.

The dataset contains **3,900 purchase records and 18 features**, enabling analysis of demographics, purchase patterns, and customer behavior. 

---

## Dataset

The dataset contains transactional records of customer purchases across multiple product categories.

**Dataset Highlights**

* **Rows:** 3,900 purchase records
* **Columns:** 18 features
* **Missing Values:** 37 values in the *Review Rating* column (handled during cleaning)

**Key Data Features**

* Customer demographics: Age, Gender, Location, Subscription Status
* Purchase details: Item Purchased, Category, Purchase Amount, Season, Size, Color
* Shopping behavior: Discount Applied, Promo Code Used, Previous Purchases, Frequency of Purchases, Review Rating, Shipping Type

This dataset allows analysis of **customer segmentation, revenue patterns, product popularity, and purchasing behavior**.

## Project Workflow

### 1. Data Loading

The dataset was imported into Python using **Pandas** for preprocessing and exploration.

### 2. Exploratory Data Analysis (EDA)

Initial exploration included:

* Dataset structure inspection using `df.info()`
* Statistical summary using `df.describe()`
* Understanding distributions and patterns in customer behavior

### 3. Data Cleaning

Several preprocessing steps were performed:

* Missing values in **Review Rating** were imputed using the **median rating per product category**
* Column names were standardized to **snake_case**
* Created new features such as:

  * `age_group`
  * `purchase_frequency_days`
* Removed redundant column **promo_code_used**

The cleaned dataset was then exported to a SQL database for further analysis. 

---

## SQL Analysis

SQL queries were used to answer key business questions such as:

* Revenue generated by **gender**
* Identifying **high-spending customers who used discounts**
* Finding **top rated products**
* Comparing **average purchase amount by shipping type**
* Analyzing **subscribers vs non-subscribers spending**
* Identifying **discount-dependent products**
* **Customer segmentation** (New, Returning, Loyal)
* **Top products per category**
* **Repeat buyers vs subscription likelihood**
* **Revenue contribution by age group**

These queries helped extract **business insights from transactional data**.

Key dashboard components include:

* Number of customers

* Customer subscription distribution

The dashboard allows stakeholders to quickly explore trends and customer behavior patterns. 

---

## Key Results & Insights

Some of the major insights discovered include:

* **Male customers generated higher revenue** compared to female customers.
* **Express shipping users spend slightly more** on average than standard shipping users.
* **Young adults contribute the highest revenue** among all age groups.
* Certain products such as **Hats and Sneakers are highly discount-dependent**.
* The majority of customers fall into the **Loyal customer segment**.

These insights can help businesses optimize marketing strategies and improve customer engagement.

* Optimize **discount strategies** to balance sales and profitability
* Highlight **top-rated products** in marketing campaigns
* Focus marketing efforts on **high-value customer segments**

## How to Run the Project

### 1. Clone the Repository

```
git clone <[repository-link](https://github.com/parthphoenix/customer_behavior_analysis)>
```

### 2. Install Required Python Libraries

```
pip install pandas numpy matplotlib seaborn psycopg2
```

### 3. Run the Python Notebook

Open and run the Jupyter notebook:

```
Customer_Shopping_Behavior_Analysis.ipynb
```

This will:

* Load the dataset
* Perform EDA
* Clean the data

### 4. Load Data into SQL Database

Run the SQL script:

```
customer_behavior_analysis.sql
```

This script performs the analytical queries.

### 5. Open the Power BI Dashboard

Open the Power BI file:

```
customer_behavior.pbix
```

### 6. View the Presentation

Open the project presentation:

```
Customer-Shopping-Behavior-Analysis.pptx
```
